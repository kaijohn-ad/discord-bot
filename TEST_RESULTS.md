## テストの実行方法

```bash

npm test

# 一度だけ実行
npm run test:run

# カバレッジ付き（@vitest/coverage-v8が必要）
npm run test:run -- --coverage
```

## 注意事項

- LLM APIキーが設定されていない場合、自然言語パーサーの一部のテストは警告を表示しますが、テストは正常に実行されます
- データベーステストは実際のSQLiteデータベースを使用します（テスト用の分離されたDBの使用を推奨）
